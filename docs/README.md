# Documentation Directory

This directory contains organized documentation for the LinuxONE RAG Pipeline project.

## 📁 Directory Structure

```
docs/
├── implementation/     # Implementation history and phase documentation
├── plans/             # Planning documents and upgrade plans
├── troubleshooting/   # Troubleshooting guides and fixes

```

- **CONTEXT_FIRST_HYBRID_RAG.md** - Hybrid RAG transition

### Planning Documents (`plans/`)
Strategic planning and upgrade documents:

- **hybrid_rag_plan.md** - Hybrid RAG architecture upgrade plan
- **linuxone_llm_architecture_upgrade_plan.md** - LLM architecture improvements
- **linuxone_ui_upgrade_plan.md** - UI/UX enhancement plans
- **ui_ux_upgrade.md** - Detailed UI upgrade specifications
- **RETRIEVAL_IMPROVEMENT_PLAN.md** - Retrieval enhancement strategy
- **RAG_DIAGNOSIS_ANALYSIS.md** - System issue diagnosis

### Troubleshooting (`troubleshooting/`)
